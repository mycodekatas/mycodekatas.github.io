one code base

With the modern web, it is possible to build a single code base capable of satisfying all platforms. It's no longer necessary to build a separate code base for the web, iOS, Android, and Windows Phone.

The MashupJS is a bootstrap for building one code base.

Get Started »

large application

The MashupJS provides the components and a prescriptive guide that can be followed. Some of the components you will find are logging, instrumentation, advanced caching, structured for large applications, Lazy loading of components. There are many more features, to many to list here.

learn teach

The large application nature of the Mashup makes it easy to build modules around technologies for demonstration and study. These modules be used for teaching and learning while hidden by security. They can serve as documentation for approaches used by whatever application is created from the Mashup.
